Hi my name is Lynn Worthy and I have some advice for you!

Be careful programming for the Web using Windows!

I have noticed on several occassions that some of the commands and techniques react differently when you use Windows

The Mac seems to be less trouble because you are using Unix style commands.

Google can be your friend

I have asked questions in Google and I have found that most of the time others have experienced the same problem

Usually there have already been solutions to the problems you are experiencing

So don't get disheartend, just hone your troubleshooting skills and your will be Ok.

Lea Camara's Advice



Be Brave!

Deciding to take a leap and learn something new - especially when you're hoping to be able to make a new career out of it - is scary!!! So - be brave. Be very brave.

Build yourself up. Repeat positive mantras.

Tell yourself you're capable. Tell yourself you can do it. Tell yourself you WILL make this happen. If they can do it, so can you!

Keep reminding yourself: You've come so far!

I know I've learned A LOT in a short amount of time. But - when there's always so much more to learn - it can feel like you're getting nowhere! That you'll NEVER be "qualified"! But - like so many other things in life (think: wealth, status, looks), we'll always be somewhere in the middle. Not the best. But definitely not the worst.

Be OK with the fact that you'll always make mistakes.

But always be humble enough to want to learn what they are and how to fix them.

Definitely keep learning.

Just don't stop.

Frame it like an adventure.

You're on an adventure to keep exploring and finding out the type of work you really enjoy.

Hi my name is Sandy and I have some advice for you!

Start with small steps.

When you start learning programming, my suggestion would be to learn something new everyday but not too much at one time, otherwise it will be too overwhelming

Learn first the basics.

Start your programming career with learning the very basic HTML and CSS. When you can code a website with these tools you can continue to learn other programming languages.

Allow yourself to be a beginner

Don't be hard on yourself. If you do not understand something, walk away from the computer for 10 minutes. When you return you're likely to catch the see the problem with fresh eyes.

Allow yourself room to fail.

Make friends.

Everything is easier with friends. Coding buddies are the best!

you will learn a lot more if working on a problem together.

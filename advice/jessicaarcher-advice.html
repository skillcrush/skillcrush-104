Hi my name is Jessica and I have some advice for you!

It's Okay!

When I first started programming I felt so lost, I started to pick it up and than things got easy. . .Until I started learning a different coding language.

I struggled and when I would mess something up I would feel defeated and annoyed that I could figure out what I was doing wrong.

What I have learned is that the coding community is great, if I have questions it is so easy to get help because there are people out there that love to help.

So take on some challenges, if you get stuck just ask for help and do some researching!

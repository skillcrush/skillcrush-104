Hi my name is Jake and I have some advice for you!

Just do, and do, and do

Learning to code is learning a new language, only its disguised because you can write english words. Take time, pay attention to details, write everything down twice. You cant learn spanish or german in a few days, coding is no different. Commit it all to muscle memory. And dont be too hard on yourself.

Hi, my name is Sophie and I've got some advice for all the code learners out there!

Learning isn't a straight path

Out of the blue you'll go from knowing the last topic inside out and feeling like you're on trac to being utterly lost.

That's ok, and the process of trying to learn where you're going wrong probably helps you retain the info better in the long run.

Nothing is perfect

There isn't always one correct anserr to a problem, someone else's code might not look exactly like yours. Learn from everyone else and use their code to see how they approached the same problem.

The race was won by the turtle, not the hare.

My name is Liz and I'm here to share some of the great wisdom I have discovered since embarking on my journey

Easy = Bad & Hard = Good

First thing to remember...if it's easy that means you're doing well but it WILL get hard so take this as a good sign to start and a bad sign after awhile.

If it's hard - good. It means you're making all the mistakes you can while learning. This is the key to development. If it wasn't hard, then it wouldn't be worth it in the end.

Run with it!

Whether every time you touch the computer something goes wrong or it goes really right, go with it!

The learning process is that - a process. One that we all need to work on everyday in order to make the changes we want.

Ask for help - it makes all the difference in the world!

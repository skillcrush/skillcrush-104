Newbie coding tips from Antonella

1. Take breaks!

When you are stuck with code that doesn't work and you've exhausted researching and experimenting, TAKE A BREAK. There have been many times I'm still bashing away at the code for hours until I finally just give up and go to bed at 1 am. Next morning I try again and figure it out in a matter of minutes. So if you are having trouble, take a walk, do something else, go to sleep and go back to the code with a fresh perspective.

2. Inspect Element is your friend

Use the Inspect Element / Developer tools to experiment and then save in your CSS. This will save lots of time.

3. Be patient and specific

Focusing on specific tasks will help you learn faster and more deeply. Have patience to research options and then experiment.

4. Pat yourself on the back

Make sure to acknowledge each new piece of code you get to work. Doesn't it feel good?
